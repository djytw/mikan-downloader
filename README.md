# mikan-downloader

Keep your anime updated with [Mikan Project](https://mikanani.me/) RSS.

## Usage

1. Install aria2c & enable jsonrpc
2. Download/clone the repo
3. Edit config.json
4. `npm i`
5. `node dist/main.js`
